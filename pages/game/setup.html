How to Place Ships
  1. Select Carrier from the left side of the screen by clicking it. It will highlight when selected.
  2. Select a square from the table to place the front/bow of the ship.
  3. Select a highlighted square from the table to place the rear/stern of the ship.
  4. Repeat with the rest of the ships from top to bottom until all are placed. Press Reset if you would like to start over.
  5. Press Ready when done!
